# Cosmoria

Open-source backend engine for building multi-tenant applications on PostgreSQL.

Cosmoria is a backend infrastructure layer that lets you build SaaS applications with dynamic APIs, multi-tenancy, and S3-compatible storage.

## Features

- Multi-tenant architecture
- Dynamic API generation from schemas
- PostgreSQL-first design
- S3-compatible storage integration
- Authentication & RBAC system
